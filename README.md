# 天幕行者 · 智能悬浮随行伞


> 解放双手，智能随行 —— 让雨伞/阳伞像忠诚的伙伴一样悬浮跟随。

---

## 📌 项目简介
**天幕行者**是一款结合无人机技术与AI视觉跟随的智能伞具。它彻底解决了雨天/烈日下撑伞带来的不便：
- 无法使用手机
- 手提物品时手忙脚乱
- 情侣/朋友间“谁撑伞”的尴尬
- 商务人士接打电话的束缚

产品通过AI识别主人位置，自动悬停跟随，并可根据雨量/紫外线调整伞面角度。让出行真正从容。

---

## 🎯 痛点与市场
- **用户痛点**：现有伞具需要用手支撑，限制了双手自由。
- **市场规模**：全球伞具市场每年超百亿美元，而智能穿戴设备正在高速增长。本产品切入“个人低空智能防护”新品类，瞄准都市白领、科技潮人、户外爱好者。

---

## 🛠 技术构想
### 核心模块
| 模块 | 技术方向 | 状态 |
|------|---------|------|
| AI视觉跟踪 | YOLO+DeepSORT 多目标跟踪 | 算法仿真中 |
| 飞控系统 | 定制Pixhawk开源飞控 | 选型中 |
| 动力系统 | 4轴碳纤维桨叶+高效无刷电机 | 供应商接洽中 |
| 电池 | 高能量密度锂电池+智能BMS | 目标续航：智能模式下可用一周 |
| 结构 | 碳纤维骨架+航空级超薄涂氟尼龙伞面 | 材料测试中 |

### 主要功能
- ✅ 手势/APP唤醒，自动起飞跟随
- ✅ 实时避障（超声波+视觉）
- ✅ 智能跟随（识别主人位置）
- ✅ 环境自适应（雨量/紫外线）
- ✅ 低电量自动返航
